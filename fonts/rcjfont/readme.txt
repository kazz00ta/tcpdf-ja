                          RCJFONT for TCPDF

This package includes fonts for TCPDF, based on RCJFONT.

= Available fonts =

This package has following fonts.

RCJFONT Gothic: rcjfontgothic.php, rcjfontgothic.z, rcjfontgothic.ctg.z
RCJFONT PGothic: rcjfontpgothic.php, rcjfontpgothic.z, rcjfontpgothic.ctg.z
RCJFONT Mincho: rcjfontmincho.php, rcjfontmincho.z, rcjfontmincho.ctg.z
RCJFONT PMincho: rcjfontpmincho.php, rcjfontpmincho.z, rcjfontpmincho.ctg.z
RCJFONTtk Gothic: rcjfonttkgothic.php, rcjfonttkgothic.z, rcjfonttkgothic.ctg.z
RCJFONTtk PGothic: rcjfonttkpgothic.php, rcjfonttkpgothic.z, rcjfonttkpgothic.ctg.z
RCJFONTtk Mincho: rcjfonttkmincho.php, rcjfonttkmincho.z, rcjfonttkmincho.ctg.z
RCJFONTtk PMincho: rcjfonttkpmincho.php, rcjfonttkpmincho.z, rcjfonttkpmincho.ctg.z

= Installation =

1.Upload all *.php, *.z, *.ctg.z files to /tcpdf/fonts/ directory.
2.Set as defaut font using setHeaderFont(), setFooterFont(), SetDefaultMonospacedFont, SetFont() etc.
3.If you allow persons who receive a PDF to make changes to it even if they don't have the same font, set "setFontSubsetting" to "false" using setFontSubsetting().

Eachs font have following font name.

RCJFONT Gothic: rcjfontgothic
RCJFONT PGothic: rcjfontpgothic
RCJFONT Mincho: rcjfontmincho
RCJFONT PMincho: rcjfontpmincho
RCJFONTtk Gothic: rcjfonttkgothic
RCJFONTtk PGothic: rcjfonttkpgothic
RCJFONTtk Mincho: rcjfonttkmincho
RCJFONTtk PMincho: rcjfonttkpmincho

                          RCJフォント for TCPDF

このパッケージには、RCJフォントをTCPDF用に変換したフォントが収められています。

= フォントの種類 =

このパッケージには、次のフォントが収められています。

RCJFONT ゴシック：rcjfontgothic.php、rcjfontgothic.z、rcjfontgothic.ctg.z
RCJFONT Pゴシック：rcjfontpgothic.php、rcjfontpgothic.z、rcjfontpgothic.ctg.z
RCJFONT 明朝：rcjfontmincho.php、rcjfontmincho.z、rcjfontmincho.ctg.z
RCJFONT P明朝：rcjfontpmincho.php、rcjfontpmincho.z、rcjfontpmincho.ctg.z
RCJFONTtk ゴシック：rcjfonttkgothic.php、rcjfonttkgothic.z、rcjfonttkgothic.ctg.z
RCJFONTtk Pゴシック：rcjfonttkpgothic.php、rcjfonttkpgothic.z、rcjfonttkpgothic.ctg.z
RCJFONTtk 明朝：rcjfonttkmincho.php、rcjfonttkmincho.z、rcjfonttkmincho.ctg.z
RCJFONTtk P明朝：rcjfonttkpmincho.php、rcjfonttkpmincho.z、rcjfonttkpmincho.ctg.z

= インストール =

1.*.php、*.z、*.ctg.zという名前のファイルを全て、「/tcpdf/fonts/」ディレクトリにアップロードします。
2.setHeaderFont()、setFooterFont()、SetDefaultMonospacedFont、SetFont()などを使って、デフォルトのフォントとして設定します。
3.PDFを受け取った相手が、同じフォント持っていなくとも編集出来るようにするには、setFontSubsetting()を使って「setFontSubsetting」を「false」に設定します。

各フォントの名前は次の通りです。

RCJFONT ゴシック: rcjfontgothic
RCJFONT Pゴシック: rcjfontpgothic
RCJFONT 明朝: rcjfontmincho
RCJFONT P明朝: rcjfontpmincho
RCJFONTtk ゴシック: rcjfonttkgothic
RCJFONTtk Pゴシック: rcjfonttkpgothic
RCJFONTtk 明朝: rcjfonttkmincho
RCJFONTtk P明朝: rcjfonttkpmincho

= ライセンス =

RCJFONT ゴシック、RCJFONT Pゴシック、RCJFONT 明朝、RCJFONT P明朝、RCJFONTtk ゴシック、RCJFONTtk Pゴシック、RCJFONTtk 明朝、RCJFONTtk P明朝には「IPAフォントライセンスv1.0」が適用されます。

詳しくは、「IPA_Font_License_Agreement_v1.0.txt」をお読み下さい。
